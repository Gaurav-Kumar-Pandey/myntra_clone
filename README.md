 # 🛍️ MYNTRA CLONE
📌 PROJECT DESCRIPTION
This project is a Myntra Clone built using HTML and CSS, designed to replicate the UI/UX of Myntra’s homepage. It features a responsive layout, interactive elements, and a user-friendly interface. This project is for educational purposes and showcases front-end development skills.

## 🚀 FEATURES
Responsive Navbar with category links
Search Bar for product search simulation
Action Bar with profile, wishlist, and cart icons
Dynamic Banner Section highlighting offers
Shop by Category section with product images
Structured Footer with important links

##m 🎨 TECHNOLOGIES USED
HTML for structuring the content
CSS for styling and layout
Google Fonts & Icons for enhanced UI
